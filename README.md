# face-detection
Face detection with Face Api

A facial expression detection app in the browser using:

>1.) WebCam
>2.) Canvas
>3.) Face API

This code base is able to detect the following facial expressions:

1.) Age Estimate
2.) Gender
3.) Emotional State
    i.)   anger
    ii.)  disgusted
    iii.) fearful
    iv.)  happy
    v.)   neutral
    vi.)  sad
    vii.) surprised
